# About my library

This library has 2 modules: math and pygame_copy(recommend not using). Module
`math` has two classes: `square_root` and `Calculator`.


CLASS Calculator:
```py
import python_mini
oCalculator = python_mini.math.Calculator(144, 12)
CalcDiv = oCalculator.div()
print(CalcDiv)  #Enter: 12.0
```


class square_root:
```py
import python_mini
sq_root = square_root(144).square_root()
print(sq_root)
```
